# Matt's Portfolio

This is the source code for my personal portfolio website. It highlights projects, skills, and social links to help potential employers and collaborators learn more about me.

## Structure
- **index.html**: Main entry point of the website.
- **styles.css**: Styling for the layout and design.
- **app.js**: Adds interactivity and animations.
- **assets/**: Placeholder for images or datasets.

## Features
- Graph animations
- Social media links
- Minimalistic design

## Deployment
This project is hosted using GitHub Pages.
